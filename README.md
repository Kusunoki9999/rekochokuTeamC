# インターンを終えて
## 良かった点(メンターによるFBを含む)
・チーム内の役割分担がハッキリしていた。
・区切りごとに残り時間を意識して動けていた。
・発表が堂々としていた。
・質疑応答でもそれほどズレた回答はしていなかった。

## 反省点(メンターによるFBを含む)
・お互いの作業状況の把握が甘かった。
  ⇒ドキュメントなどで適宜共有
・メンターへの質問が少なかった。不明点のまま時間が過ぎるくらいなら積極的に質問をする。
・ユーザーストーリーから逆算して実装内容の優先順位を決める。
